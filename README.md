Analysing Electric Grid Data
=======
This project analyzes hourly electric load data across different regions of India from Jan 1 2019 to Apr 30 2024.
It is based on the Kaggle Dataset: https://www.kaggle.com/datasets/shubhamvashisht/hourly-load-india-electrical-load-forecasting
>This data is obtained from government portals as said by the creator, yet it is NOT independently verified by me.


It offers practical insights on the evolution of the demand of power over time.
It was made using:
 - Pandas
 - Numpy
 - MatPlotLib
